# OrkestrAI Referential (Open Source)

Open-source, multilingual repository of the OrkestrAI certification referential.

## Purpose

This repository hosts the public referential used to evaluate AI-assisted software development orchestration quality.

Goals:
- maintain a transparent and versioned standard;
- allow community contributions through pull requests;
- keep editorial validation under OrkestrAI maintainers.

## Languages and versions

Current structure:

- `content/fr/v1/referentiel.md`
- `content/en/v1/referentiel.md`
- `content/fr/v1/changelog.md`
- `content/en/v1/changelog.md`

Future versions should use `v2`, `v3`, etc. without rewriting history.

## Contribution model

Contributions are welcome through pull requests.

Main branch protections:
- direct push to `main` is blocked;
- pull request review is required;
- CODEOWNERS approval is required for merge.

See [CONTRIBUTING.md](./CONTRIBUTING.md).
