# OrkestrAI Referential Standard v1

## Status

Version: `v1`  
Source language: `fr`  
Last updated: `2026-03-02`

## Objective

The OrkestrAI referential evaluates the quality of human orchestration in AI-assisted software development.

It notably covers:
- architecture and decoupling;
- robustness and testing;
- application security;
- decision traceability;
- long-term maintainability.

## Evaluation axes (v1 structure)

1. Software architecture and design
2. AI orchestration and context mastery
3. Code quality, testing, and robustness
4. Security, compliance, and risk management
5. Operability, documentation, and technical governance

## Scoring

- Total: `100 points`
- Criterion score: `0` to `4`
- Certification level based on global score

## Contribution rules on v1

- Substantive changes must keep v1 semantic compatibility.
- Changes that alter evaluation logic should target a future version (`v2`).
